# react-native-lf-thread

Asynchronous Threading Tool Library for react-native

## Installation

```sh
npm install react-native-lf-thread
```

## Usage


```js
import { multiply } from 'react-native-lf-thread';

// ...

const result = multiply(3, 7);
```
